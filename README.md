# MapAI

MapAI is a map generation tool that starts with CSV-to-map creation and grows toward AI chat-based map building.

## Current Version

- `v1.0`: CSV to OpenStreetMap HTML generation with Streamlit app support.

## Roadmap

| Version | Goal | Status |
| --- | --- | --- |
| v1.0 | CSV to map generation | Ready |
| v1.1 | Streamlit publishing polish | Planned |
| v1.2 | Filter and search | Planned |
| v2.0 | AI chat map creation | Planned |
| v3.0 | Google Maps API support | Planned |

## Deploy v1.0 to Streamlit Community Cloud

1. Push this `MapAI` folder to GitHub.
2. Open Streamlit Community Cloud.
3. Create a new app from the GitHub repository.
4. Set Main file path to `v1.0/streamlit_app.py`.
5. Deploy the app and share the generated URL.

If you put only the contents of `v1.0` at the repository root, set Main file path to `streamlit_app.py`.
